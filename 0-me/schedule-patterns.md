# Schedule Patterns

Fill in your fixed weekly blockers here. The calendar scheduling skill reads this file
when reasoning about your availability.

Examples to add:
- Office days (days you commute and aren't available for deep-focus sessions)
- Custody windows or family-protected evenings
- Fixed recurring commitments (classes, therapy, workouts, etc.)
- Deep-work windows you want the system to protect

This file is consulted only when the calendar integration is enabled.
